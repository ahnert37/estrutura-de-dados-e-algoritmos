# Linked List
- Ele é diferente do array, não é sequencial na memória igual o array.

### Linkedin List vs Array

- Leitura
  - Array: O(1)
  - Linked List: O(n)

- Inserção
  - Array: O(n)
  - Linked List: O(1)

___

- Linked List vai se comportar melhor com o primeiro e o último elemento O(1). O restante das operações vão ser O(n).
